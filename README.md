## Dependencies

- Node.js 16+
- npm 8+

## Run Locally

Clone the project

```bash
  git clone https://github.com/OddBlueDog/backend-news-api
```

Go to the project directory

```bash
  cd backend-news-api
```

Install dependencies

```bash
  npm install
```

Create env file (.env) in root containing news api key

```
NEWS_API_KEY="KEY_HERE"
```

Start the server

```bash
  npm run nodemon
```
